# MikizBack

REST API backend for Mikiz — a platform hosting French mini games. Built with Bun, Hono, PostgreSQL, and Drizzle ORM.

---

## Tech Stack

| Layer | Tool |
|---|---|
| Runtime | Bun ≥ 1.1 |
| Framework | Hono 4 |
| Database | PostgreSQL 15+ via `postgres` driver |
| ORM | Drizzle ORM + Drizzle Kit |
| Auth | JWT (jose) |
| Linter/Formatter | Biome |
| Tests | Bun test |

---

## Local Setup

### 1. Prerequisites

- [Bun](https://bun.sh) ≥ 1.1
- PostgreSQL 15+

### 2. Install dependencies

```bash
bun install
```

### 3. Configure environment

```bash
cp .env.example .env
# Edit .env — set DATABASE_URL and JWT_SECRET
```

### 4. Run migrations

```bash
bun db:generate   # generate SQL from schema (first time or after schema changes)
bun db:migrate    # apply migrations to the database
```

### 5. Seed initial data

```bash
bun db:seed                # creates game row + today's test word (MONDE)
bun words:schedule 60      # schedule 60 days of words from today
bun words:schedule 30 2026-06-01  # schedule from a specific start date
```

### 6. Start the server

```bash
bun dev    # watch mode
bun start  # production
```

The API runs on `http://localhost:3000` by default.

### Sutom

#### `GET /api/sutom/daily` — public

Returns today's word metadata without revealing the word.

```json
// 200
{ "date": "2026-05-15", "wordLength": 5, "firstLetter": "M" }
```

#### `GET /api/sutom/session` — protected

Returns the user's current session for today, including all guesses made so far.

```json
// 200 — in progress
{
  "session": {
    "status": "in_progress",
    "attempts": [
      { "guess": "FLEUR", "result": ["absent", "absent", "absent", "absent", "present"] }
    ],
    "wordLength": 5,
    "firstLetter": "M"
  }
}

// 200 — game over (word is revealed)
{
  "session": {
    "status": "won",
    "attempts": [...],
    "wordLength": 5,
    "firstLetter": "M",
    "word": "MONDE"
  }
}

// 200 — no session yet
{ "session": null }
```

#### `POST /api/sutom/guess` — protected

```json
// Body
{ "guess": "monde" }  // case-insensitive

// 200 — in progress
{
  "result": {
    "guess": "MONDE",
    "result": ["correct", "correct", "correct", "correct", "correct"]
  },
  "status": "won",
  "attemptsLeft": 5,
  "word": "MONDE"
}

// Letter result values:
//   "correct"  — right letter, right position (green)
//   "present"  — right letter, wrong position (yellow)
//   "absent"   — letter not in word (grey)
```

| Status | Meaning |
|---|---|
| `in_progress` | Game continues |
| `won` | User guessed correctly |
| `lost` | 6 attempts exhausted without a correct guess |

**Error codes:**
- `400` — guess missing or wrong length
- `404` — no word scheduled today
- `409` — game already completed
- `422` — not a valid French word

### Leaderboard

#### `GET /api/leaderboard/:game?date=YYYY-MM-DD` — public

Returns today's leaderboard (or a specific date's) for a given game.
Wins are sorted by fewest attempts; losses appear at the bottom.

```json
// GET /api/leaderboard/sutom?date=2026-05-15
{
  "date": "2026-05-15",
  "game": "sutom",
  "total": 3,
  "entries": [
    { "username": "alice", "score": 2, "completedAt": "2026-05-15T08:12:00Z" },
    { "username": "bob",   "score": 5, "completedAt": "2026-05-15T09:40:00Z" },
    { "username": "carol", "score": null, "completedAt": "2026-05-15T10:05:00Z" }
  ]
}
```

`score` = number of attempts to win. `null` = lost.

## Sutom Rules

- One French word per day, same for all users.
- 6 attempts maximum.
- The **first letter is always revealed** to the player at the start (classic Sutom rule).
- Letter feedback: `correct` / `present` / `absent`.
- The word is revealed once the game ends (win or loss).

## French Word List

The sample `words/fr-words.txt` is for local development only. For production, replace it with a curated list. Recommended sources:

- **[Lexique](http://www.lexique.org/)** — `Lexique383.zip` contains a large French lexicon. Extract the 5–9 letter nouns.
- **[DELA / Unitex](https://unitexgramlab.org/)** — linguistic dictionaries with POS tags, filterable by category.
- A hand-curated list of common French nouns (most Wordle clones use ~2 000–5 000 words).

The word loader (`src/lib/words.ts`) accepts any plain-text file with one uppercase word per line, 5–9 letters, no accents. Words outside that range are silently skipped.

## Adding a New Game

1. Add a new table in `src/db/schema/` (e.g., `spotle.ts`).
2. Add its game logic to `src/lib/`.
3. Create `src/routes/<game>.ts` following the Sutom pattern.
4. Mount it in `src/index.ts`.
5. Seed the new `games` row via `scripts/seed.ts`.
6. Generate and run migrations: `bun db:generate && bun db:migrate`.
