# MikizBack

REST API backend for Mikiz — a platform hosting French mini games. Built with Bun, Hono, PostgreSQL, and Drizzle ORM.

---

## Tech Stack

| Layer | Tool |
|---|---|
| Runtime | Bun ≥ 1.1 |
| Framework | Hono 4 |
| Database | PostgreSQL 15+ via `postgres` driver |
| ORM | Drizzle ORM + Drizzle Kit |
| Auth | JWT (jose) |
| Linter/Formatter | Biome |
| Tests | Bun test |

---

## Local Setup

### 1. Prerequisites

- [Bun](https://bun.sh) ≥ 1.1
- PostgreSQL 15+
- A [TMDB API key](https://www.themoviedb.org/settings/api) (for the film game)

### 2. Install dependencies

```bash
bun install
```

### 3. Configure environment

```bash
cp .env.example .env
# Edit .env — set DATABASE_URL, JWT_SECRET, and TMDB_API_KEY
```

### 4. Run migrations

```bash
bun db:generate   # generate SQL from schema (first time or after schema changes)
bun db:migrate    # apply migrations to the database
```

### 5. Seed initial data

```bash
bun db:seed                          # creates game rows
bun words:schedule 60                # schedule 60 days of words from today
bun words:schedule 30 2026-06-01     # schedule words from a specific start date
bun films:schedule 30                # schedule 30 days of films from today
```

### 6. Start the server

```bash
bun dev    # watch mode
bun start  # production
```

The API runs on `http://localhost:3000` by default.

### Motivex

#### `GET /api/motivex/daily` — public

Returns today's word metadata without revealing the word.

```json
// 200
{ "date": "2026-05-15", "wordLength": 5, "firstLetter": "M" }
```

#### `GET /api/motivex/session` — protected

Returns the user's current session for today, including all guesses made so far.

```json
// 200 — in progress
{
  "session": {
    "status": "in_progress",
    "attempts": [
      { "guess": "FLEUR", "result": ["absent", "absent", "absent", "absent", "present"] }
    ],
    "wordLength": 5,
    "firstLetter": "M"
  }
}

// 200 — game over (word is revealed)
{
  "session": {
    "status": "won",
    "attempts": [...],
    "wordLength": 5,
    "firstLetter": "M",
    "word": "MONDE"
  }
}

// 200 — no session yet
{ "session": null }
```

#### `POST /api/motivex/guess` — protected

```json
// Body
{ "guess": "monde" }  // case-insensitive

// 200 — in progress
{
  "result": {
    "guess": "MONDE",
    "result": ["correct", "correct", "correct", "correct", "correct"]
  },
  "status": "won",
  "attemptsLeft": 5,
  "word": "MONDE"
}
```

Letter result values: `"correct"` (right position), `"present"` (wrong position), `"absent"` (not in word).

| Status | Meaning |
|---|---|
| `in_progress` | Game continues |
| `won` | User guessed correctly |
| `lost` | 6 attempts exhausted |

**Error codes:** `400` wrong length, `404` no word today, `409` already completed, `422` not a valid French word.

### Film du Jour (CinéClue)

#### `GET /api/filmdujour/session` — protected

Returns the user's current film game state for today.

```json
// 200 — in progress
{
  "session": {
    "status": "in_progress",
    "tentatives": [
      { "tmdbId": 11, "titre": "Star Wars" }
    ],
    "indices": {
      "genres": ["Science-Fiction"],
      "pays": [],
      "acteurs": [],
      "realisateur": null,
      "annee": { "min": 1970, "max": 1990 },
      "duree": null,
      "langue": null
    }
  }
}

// 200 — game over (film is revealed)
{
  "session": {
    "status": "won",
    "tentatives": [...],
    "indices": {...},
    "filmCible": {
      "id": 238,
      "titre": "Le Parrain",
      "annee": 1972,
      "duree": 175,
      "genres": ["Drame", "Crime"],
      "pays": ["États-Unis"],
      "langue": "en",
      "realisateurs": ["Francis Ford Coppola"],
      "acteurs": ["Marlon Brando", "Al Pacino", ...]
    }
  }
}

// 200 — no session yet
{ "session": null }
```

#### `POST /api/filmdujour/guess` — protected

```json
// Body
{ "tmdbId": 238 }

// 200
{
  "tentative": { "tmdbId": 238, "titre": "Le Parrain" },
  "indices": {
    "genres": ["Drame"],
    "pays": ["États-Unis"],
    "acteurs": [],
    "realisateur": null,
    "annee": { "min": 1965, "max": 1985 },
    "duree": { "min": 150, "max": 200 },
    "langue": null
  },
  "status": "in_progress",
  "tentativesLeft": 9
}
```

**Error codes:** `400` missing tmdbId, `404` film not found / no film today, `409` already completed.

### CinéClue Search

#### `GET /api/cineclue/search?q=<query>` — public

Autocomplete film search via TMDB. Requires at least 3 characters.

```json
// GET /api/cineclue/search?q=parrain
// 200
{
  "results": [
    { "id": 238, "titre": "Le Parrain", "annee": 1972, "poster": "/path.jpg" },
    ...
  ]
}
```

Returns up to 20 results. `503` if TMDB is unreachable.

### Leaderboard

#### `GET /api/leaderboard/:game?date=YYYY-MM-DD` — public

Returns today's (or a specific date's) leaderboard for `motivex` or `cineclue`.

```json
// GET /api/leaderboard/motivex?date=2026-05-15
{
  "date": "2026-05-15",
  "game": "motivex",
  "total": 3,
  "entries": [
    { "username": "alice", "score": 2, "completedAt": "2026-05-15T08:12:00Z" },
    { "username": "bob",   "score": 5, "completedAt": "2026-05-15T09:40:00Z" },
    { "username": "carol", "score": null, "completedAt": "2026-05-15T10:05:00Z" }
  ]
}
```

`score` = number of attempts (lower is better). `null` = lost.

#### `GET /api/leaderboard/cross?date=YYYY-MM-DD` — public

Cross-game leaderboard combining all games using F1 scoring (1st→25pts, 2nd→18pts, … loss→0pts).

#### `GET /api/leaderboard/:game/stats` and `GET /api/leaderboard/cross/stats` — public

All-time aggregate stats per game or across all games.

## Game Rules

### Motivex

- One French word per day, same for all users.
- 6 attempts maximum.
- The first letter is always revealed at the start.
- Letter feedback: `correct` / `present` / `absent`.

### CinéClue

- One film per day, same for all users.
- 10 attempts maximum.
- Each wrong guess reveals partial hints: overlapping genres, countries, actors, and ranges for year and duration.
- The target film is revealed when the game ends (win or loss).

## French Word List

The `words/fr-daily-words.txt` and `words/fr-valid-words.txt` files drive the Motivex game. Recommended sources:

- **[Lexique](http://www.lexique.org/)** — `Lexique383.zip` contains a large French lexicon.
- **[DELA / Unitex](https://unitexgramlab.org/)** — linguistic dictionaries with POS tags.

The word loader (`src/lib/words.ts`) accepts plain-text files with one uppercase word per line, 5–9 letters, no accents.

## Adding a New Game

1. Add a new table in `src/db/schema/` (e.g., `spotle.ts`).
2. Add game logic to `src/lib/`.
3. Create `src/routes/<game>.ts` following the Motivex or CinéClue pattern.
4. Mount it in `src/index.ts`.
5. Seed the new `games` row via `scripts/seed.ts`.
6. Generate and run migrations: `bun db:generate && bun db:migrate`.
